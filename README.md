# ProgramacaoIProjetoFinal

Instrução de uso:
Cliente: na tela do cliente, o cliente escolhe os itens e confirma seu pedido.
Administração: Quando há um pedido no sistema, o usuário atualiza a lista e pode selecionar um pedido para ver os detalhes. Quando concluído, o usuário pode mover o pedido para lista de pedidos concluídos ao clicar o botão.
